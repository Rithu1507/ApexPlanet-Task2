## Project Overview
This project focuses on Exploratory Data Analysis (EDA) and Business Intelligence using the Sample Superstore dataset.

The dataset was analyzed to identify sales trends, profit distribution, category performance, and regional business insights.

---

## Tasks Performed

### 1. Descriptive Statistics
- Dataset summary
- Mean, median, maximum, minimum values

### 2. Univariate Analysis
- Sales distribution histogram
- Profit distribution histogram
- Category count analysis

### 3. Business Analysis
- Highest sales category
- Highest profit region
- Top 5 sales records
- Quantity sold by category

### 4. Advanced Visualization
- Scatter Plot
- Correlation Matrix
- Heatmap
- Regional Sales Analysis

### 5. Dashboard KPI Planning
- Total Sales
- Total Profit
- Total Quantity Sold
- Average Discount

---

## Key Insights
- Technology category generated high sales.
- Discounts impacted profits in several transactions.
- Regional analysis revealed differences in sales performance.
